<H1>RoboNet: A Sample Efficient Co-Design Generator</H1>
<p>RoboNet is a combination of GFlowNet with rate-based design prioritization and cost-aware sampling strategy to evaluate robots based on future promise under full training, thus, using less samples to determine best co-designs for a given task.</p>
